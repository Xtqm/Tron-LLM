## Tron: a natively agentic, LAM (Large Agentic Model)

Tron is (or will be) a model that learns to perceive, plan, act, verify, and adapt in pursuit of goals, rather than only generating text.

### Setup

1. Create or refresh the environment with your preferred Python 3.11+ toolchain.
2. Install the project dependencies.
3. Run the package entrypoint or the legacy script wrapper.

### Run

```bash
tron-llm
```

or

```bash
python main.py
```

### Notes

- The project is intentionally torch-only for now.
- The CUDA 12.6 index is configured for Windows and Linux installs of `torch`.
